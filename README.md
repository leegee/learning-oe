# Language Learning

## Features

* Easy to create lessons in JSON
* Easy to create translations in JSON
* Three types of lessons:
    1. fill in the blanks
    1. match vocabulary
    1. multiple choice

## NB

Any resemblance to other language learning apps  is purely coincidence.
